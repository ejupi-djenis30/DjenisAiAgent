# Support

## Something is broken

Open a [bug report](https://github.com/ejupi-djenis30/DjenisAiAgent/issues/new?template=bug_report.yml)
with your operating system, Python version, permission tier, the smallest safe reproduction and
the checks you already ran. Replace tokens, private paths, screenshots and logs with synthetic
values before posting.

## You want to propose a change

Use the [feature request](https://github.com/ejupi-djenis30/DjenisAiAgent/issues/new?template=feature_request.yml)
and explain the concrete workflow it would improve. A smaller proposal with a clear permission
boundary is easier to evaluate than a broad request for more autonomy.

## The report is sensitive

Do not open a public issue for a vulnerability or exposed credential. Follow the
[security policy](SECURITY.md) and use
[private vulnerability reporting](https://github.com/ejupi-djenis30/DjenisAiAgent/security/advisories/new).

This is an open-source experimental project, not a managed service. Response times are not
guaranteed, but complete and reproducible reports are prioritized.
